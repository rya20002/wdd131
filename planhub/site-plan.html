PlanHub Site Plan

An organized digital hub for recipes, lists, goals, and everyday life.

Site Name

LifeHub

This name reflects the purpose of the site: a clean and structured space where users can organize everyday life in one place. It communicates focus, calm, and simplicity.

Purpose

LifeHub is a personal lifestyle dashboard designed to help users organize different areas of their life in one structured space. The site allows users to manage recipes, track goals, save favorite activities, and build custom lists in a clean, visually appealing interface.

Audience

The intended audience is college students and young adults who want a simple and organized system for managing daily life without using several different apps. It is designed for users who value productivity, clarity, and aesthetic design.

Dynamic Elements

This site will use JavaScript to create interactive tools including a recipe section with filtering, a to-do list, a custom list builder, and a goal tracker. The user will be able to add and remove items, mark tasks complete, update goal progress, and interact with recipe cards and dashboard data.

The site will use DOM manipulation, event listeners, conditional logic, arrays, objects, and array methods such as forEach(), filter(), find(), and map().
